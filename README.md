# Honoka_JavaScriptUtils

JavaScript 常用方法的一个集合库

---
方法清单：

1. addLoadEvent：将所需函数绑定到 ```window.onload``` 事件上。
2. insertAfter：将新元素节点添加到目标元素节点后面。
3. getAjaxObject：得到 Ajax 对象。
